// Les projets fil rouge

Deux vrais outils, construits module par module

Pas des mini-exercices isolés. Deux programmes qui grandissent tout au long de la formation — et qui finissent prêts à être utilisés en production.

🖥️
gowatch
Outil CLI de monitoring système
Un binaire unique qui surveille votre machine en temps réel. Du simple affichage du Module 01 jusqu'au daemon concurrent avec scanner réseau du Module 05.
  • Collecte concurrente : CPU, RAM, goroutines
  • Métriques réelles depuis /proc sur Linux
  • Mode surveillance avec rafraîchissement configurable
  • Scanner de ports TCP concurrent
  • Export HTTPS vers gohub
  • Multi-plateforme : Linux, macOS, Windows, ARM
🌐
gohub
Dashboard API REST
Une API HTTP qui stocke et expose les métriques collectées par gowatch. Testée, dockerisée, prête pour la production en image de 8 Mo.
  • 4 endpoints REST : /health, /metrics, /history, /ingest
  • Store en mémoire thread-safe (sync.RWMutex)
  • Logs structurés JSON avec log/slog
  • Graceful shutdown sur SIGINT/SIGTERM
  • Suite de tests : Table-Driven + httptest + benchmarks
  • Dockerfile multi-stage : image finale de 8 Mo
